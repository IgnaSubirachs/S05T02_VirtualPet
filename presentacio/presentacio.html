🛸 ALIEN PET ZOO 🛸

Sistema de Gestió de Mascotes Alienígenes

Desenvolupat amb IA (v0) + Stack Modern

Aplicació full-stack per gestionar un zoo de criatures extraterrestres

🎯 Objectius del projecte

  • ⚛️ Generar el frontend amb ajuda de la IA v0
  • ☕ Implementar el backend amb Spring Boot + MySQL
  • 🐳 Desplegar l’aplicació amb Docker Compose

🤖 IA seleccionada: v0

Per al frontend vaig utilitzar v0 AI (generació de UI amb React/Next.js). La vaig escollir perquè permet:

  • ⚡ Prototipar pantalles en minuts
  • 🎨 Generar codi ja integrat amb Tailwind + shadcn/ui
  • 🚀 Accelerar el desenvolupament sense perdre qualitat
  • 🔄 Iteració ràpida amb feedback visual instantani
  • 📱 Responsive design automàtic amb breakpoints

💬 Registre d'interaccions amb v0

Exemples reals de com v0 em va guiar durant el desenvolupament:

Jo: "Crea un dashboard per gestionar mascotes alienígenes amb estadístiques"
v0: "Creo un dashboard amb grid responsive, cards per cada mascota, i gràfics d'estadístiques usando Recharts"
Jo: "Les imatges no es redimensionen bé en mòbil"
v0: "Afegeixo classes responsive: w-16 md:w-24 lg:w-32 per escalar les imatges segons el breakpoint"
Jo: "Necessito dockeritzar el projecte però no funciona l'autenticació"
v0: "El problema són les URLs. Dins Docker usa 'backend-virtualpet:8080', des del navegador 'localhost:8080'"
Jo: "Vull que els sons funcionin quan em carrego a les mascotes"
v0: "Creo una funció playReleaseSound() amb new Audio() i gestió d'errors per compatibilitat del navegador"

  • ⚛️ Frontend (Next.js) consumeix l’API REST de Spring Boot
  • 🔑 JWT per seguretat i autenticació
  • ❌ Problema: CORS → ✅ Solució: configuració al backend
  • 🧪 Proves amb Postman i integració amb Docker

⚔️ Dificultats i solucions

  • ❌ Error de connexió DB → ✅ Variables d’entorn al .env
  • ❌ Problemes amb CORS → ✅ Configuració a Spring Security
  • ❌ Integració Docker → ✅ Multi-stage build per Spring Boot

🧩 Reflexió final sobre la IA

La IA és una eina potent, però el seu ús excessiu pot convertir-nos en simples executors sense criteri propi.

El que arriba ràpid, també marxa ràpid: els coneixements apresos de manera instantània no arrelen si no hi ha pràctica i esforç personal.

La clau és utilitzar la IA com a suport per aprendre, no com a substitut de la nostra capacitat de pensar i resoldre problemes.

🧠 Reflexió sobre v0

✅ Avantatges de v0

  • Velocitat: Prototips en minuts
  • Qualitat: Codi modern i net
  • Aprenentatge: Descobreixo noves tècniques
  • Consistència: Estils uniformes
  • Responsive: Mobile-first automàtic

⚠️ Limitacions trobades

  • Context: No recorda sessions anteriors
  • Backend: No genera APIs Spring Boot
  • Personalització: Cal ajustar l'estètica
  • Lògica: Funcionalitats complexes requereixen intervenció
  • Debugging: Errors requereixen coneixement propi

Conclusió: v0 és excel·lent per accelerar el frontend, però el desenvolupador segueix sent essencial per l'arquitectura, lògica de negoci i integració amb sistemes existents.
