🛸 ALIEN PET ZOO 🛸

Sistema de Gestió de Mascotes Alienígenes

Desenvolupat amb IA (v0) + Stack Modern

Aplicació full-stack per gestionar un zoo de criatures extraterrestres

🤖 IA seleccionada: v0

Per al frontend vaig utilitzar v0 AI (generació de UI amb React/Next.js). La vaig escollir perquè permet:

  • Prototipar pantalles en minuts
  • Generar codi ja integrat amb Tailwind + shadcn/ui
  • Accelerar el desenvolupament sense perdre qualitat

💬 Registre d'interaccions amb la IA

Exemples de com la IA em va guiar:

Pregunta: “Com connecto el frontend amb el backend?”
Resposta IA: “Has de configurar els endpoints REST al backend i utilitzar fetch/axios al frontend.”
Pregunta: “Com puc dockeritzar backend i base de dades?”
Resposta IA: “Fes un docker-compose.yml amb serveis per Spring Boot i MySQL, utilitzant variables d’entorn.”

🔍 Anàlisi del codi generat per la IA

Exemple de component generat:


export default function PetCard({ pet }) {
  return (
    <div className="p-4 bg-gray-900 rounded-xl">
      <h3 className="text-xl font-bold">{pet.name}</h3>
      <p>Hunger: {pet.hunger}</p>
      <p>Status: {pet.status}</p>
    </div>
  )
}

El codi era correcte però vaig afegir:

  • Validacions addicionals
  • Connexió amb l’API backend (DTOs)
  • Traducció de textos i estètica personalitzada

🔗 Connexió Frontend ↔ Backend

  • Frontend (Next.js) consumeix l’API REST de Spring Boot
  • JWT per seguretat i autenticació
  • Problema: CORS → Solució: configuració al backend
  • Proves amb Postman i integració amb Docker

  • Com integrar frontend i backend de manera professional
  • Com aprofitar la IA per accelerar sense perdre control

🧩 Reflexió final sobre la IA

La IA és una eina potent, però el seu ús excessiu pot convertir-nos en simples executors sense criteri propi.

El que arriba ràpid, també marxa ràpid: els coneixements apresos de manera instantània no arrelen si no hi ha pràctica i esforç personal.

La clau és utilitzar la IA com a suport per aprendre, no com a substitut de la nostra capacitat de pensar i resoldre problemes.
